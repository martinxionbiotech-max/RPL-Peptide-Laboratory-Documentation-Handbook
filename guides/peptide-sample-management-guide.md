## 1. Introduction

Systematic sample management ensures every peptide sample is properly tracked, stored, and documented throughout its lifecycle.

## 2. Receiving Samples

[ ] Log receipt date
[ ] Record batch number
[ ] Verify against COA
[ ] Assign internal ID if needed
[ ] Store immediately

## 3. Registration

Create sample record with:
- Product name
- Batch number
- Quantity received
- Storage location
- Expiry/retest date

## 4. Labeling

Each container should show:
- Product name (abbreviated OK)
- Concentration (if reconstituted)
- Date prepared
- Batch number
- User initials

## 5. Storage

Organize by product or project.
Use clear, consistent labeling.
Monitor temperature regularly.
Keep desiccant in storage containers.

## 6. Tracking

Track:
- Opening date
- Aliquots removed
- Remaining quantity
- Freeze-thaw cycles
- Any quality observations

## 7. Disposal

Dispose according to institutional guidelines.
Record disposal date and method.
Update sample inventory.
