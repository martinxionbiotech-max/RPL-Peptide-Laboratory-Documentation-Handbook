## 1. Introduction

Safe and proper handling of peptides protects both the researcher and the product. This guide covers laboratory safety, contamination prevention, and proper handling protocols.

## 2. Laboratory Safety

Wear appropriate PPE:
- Lab coat
- Nitrile gloves
- Safety glasses
Work in fume hood when handling powder
Follow institutional biosafety guidelines

## 3. Receiving Materials

[ ] Inspect package integrity
[ ] Check temperature indicators
[ ] Verify product labels match packing slip
[ ] Document receipt date
[ ] Store immediately at recommended temperature

## 4. Visual Inspection

Check for:
- Discoloration (yellow/brown → possible degradation)
- Clumping or moisture in vial
- Cracked or damaged vial
- Compromised seal

## 5. Label Verification

Verify:
- Product name matches COA
- Batch/lot number matches COA
- Quantity (mg per vial)
- Storage instructions

## 6. Sample Handling

Use sterile technique
Minimize vial opening time
Work quickly to prevent moisture absorption
Use clean, dedicated equipment

## 7. Preventing Contamination

[ ] Use sterile tips and tubes
[ ] Change gloves between samples
[ ] Clean work surface before/after use
[ ] Never pipette by mouth
[ ] Use laminar flow hood for sensitive work

## 8. Documentation

Record for each peptide:
- Date received
- Date opened
- Reconstitution details
- Storage location
- Usage log

## 9. Disposal

Dispose according to institutional guidelines for biochemical waste.
Do not dispose in regular trash or down the drain.
Check local regulations for specific requirements.

## 10. Best Practices

[ ] Always label aliquots clearly
[ ] Keep peptide logbook
[ ] Monitor freezer temperature
[ ] Train all lab members on handling protocols
[ ] Report any quality concerns to RPL Peptide
