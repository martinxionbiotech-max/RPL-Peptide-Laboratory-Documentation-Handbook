# Application Note — Retatrutide

**Application Note No.:** RPL-AN-RET-001
**Date:** July 2026
**Subject:** In Vitro Research Protocols for Retatrutide

---

## 1. cAMP Accumulation Assay

Retatrutide activates GLP-1R, GIPR, and GCGR — all coupled to Gαs — leading to intracellular cAMP accumulation.

### Protocol
1. Seed receptor-expressing cells in 384-well plates
2. Add IBMX (0.5 mM) for 15 min
3. Add retatrutide serial dilutions (10⁻¹² to 10⁻⁶ M)
4. Incubate 30 min at 37°C
5. Lyse and measure cAMP per kit instructions
6. Calculate EC₅₀

### Expected EC₅₀
- GLP-1R: 0.5–5 nM
- GIPR: 0.1–2 nM
- GCGR: 1–10 nM

## 2. Beta-Arrestin Recruitment

Use PathHunter or BRET-compatible cells expressing target receptors.

## 3. Binding Affinity

Radioligand binding assay with [¹²⁵I]-GLP-1, [¹²⁵I]-GIP, or [¹²⁵I]-glucagon.

## 4. Notes
- Use low-binding tubes
- DMSO ≤0.1% final
- Triplicate, repeat ≥3 times
