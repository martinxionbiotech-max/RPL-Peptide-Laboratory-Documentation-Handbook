## 1. Introduction

Quality control ensures every batch meets established specifications. This guide describes the QC process for RPL Peptide products.

## 2. QC Overview

Each batch undergoes:
1. Incoming material inspection
2. In-process testing during synthesis
3. Final product testing
4. Batch release review

## 3. Incoming Inspection

Verify raw materials (Fmoc-amino acids, resins, reagents)
Check certificates of analysis for starting materials
Document receipt and storage

## 4. Identity Testing

HPLC retention time match
ESI-TOF MS molecular weight confirmation
Both must pass for batch release

## 5. Purity Testing

Analytical RP-HPLC at 214 nm
Standard specification: ≥98%
System suitability verified before each run

## 6. Moisture Testing

Karl Fischer titration
Specification: <5%
Higher moisture indicates handling or storage issues

## 7. Documentation Review

All QC data is compiled into the COA
Batch records are reviewed before release
Complete traceability from raw material to finished product

## 8. Batch Release

Batch released only after:
- All tests pass specifications
- Documentation is complete
- Quality manager approval

## 9. Record Keeping

QC records archived for minimum 5 years
Digital copies available through https://rplpeptides.com

## 10. Deviations

Any out-of-specification result triggers investigation.
Non-conforming batches are quarantined and rejected.

## 11. Corrective Actions

Root cause analysis for any quality issue
Process improvements implemented
Verification of effectiveness
